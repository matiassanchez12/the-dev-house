# Page Spec: About

## Objetivo

Explicar que es The Dev House, para quien esta pensado y que problema resuelve.

## Ruta y pagina

- Ruta: `/about`
- Archivo React: `resources/js/pages/public/about.tsx`
- Titulo SEO: `Acerca de | The Dev House`

## Estructura de contenido

1. Hero
   - Titulo de identidad de marca.
   - Subtitulo: comunidad para construir software en equipo.
2. Propuesta de valor
   - 3 bloques: colaborar, aprender, lanzar.
3. Mision y vision
   - Breve y concreta, tono humano.
4. Como funciona la comunidad
   - Flujo resumido: crear/descubrir proyecto, sumar equipo, iterar y publicar.
5. CTA final
   - `Ver proyectos` y `Unirme ahora`.

## Requisitos de UI

- Misma familia visual de landing (cards, iconos, separaciones).
- Evitar texto denso; priorizar secciones cortas y claras.
- Buen ritmo visual entre titulos, parrafos y bloques destacados.

## Criterios de aceptacion

- Queda claro en menos de 30 segundos que hace la plataforma.
- Mantiene coherencia con tono de marca.
- Tiene CTA accionable visible.

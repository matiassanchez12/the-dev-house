# Page Spec: Privacy

## Objetivo

Explicar de forma breve que datos se usan, para que se usan y como el usuario puede eliminar su cuenta junto con su informacion publicada.

## Ruta y pagina

- Ruta: `/privacy`
- Archivo React: `resources/js/pages/public/privacy.tsx`
- Titulo SEO: `Privacidad | The Dev House`

## Estructura de contenido

1. Introduccion
   - Compromiso con privacidad y transparencia.
2. Que datos recopilamos
   - Datos de cuenta (nombre, email, perfil).
   - Contenido publicado por el usuario (proyectos, mensajes, links).
3. Para que usamos los datos
   - Operar la plataforma, mejorar experiencia y seguridad.
4. Comparticion de datos
   - No venta de datos personales; compartir solo cuando sea necesario para operar.
5. Eliminacion de cuenta y contenido
   - El usuario puede solicitar o ejecutar eliminacion de cuenta.
   - Se elimina tambien la informacion publicada asociada, salvo obligaciones legales minimas.
6. Contacto de privacidad
   - Canal para consultas o solicitudes.

## Requisitos de UI

- Estilo documental claro, con secciones bien separadas.
- Aviso destacado para derecho de eliminacion de cuenta/informacion.
- Lenguaje simple, no legalista extremo.

## Criterios de aceptacion

- Queda explicito el uso de datos en lenguaje claro.
- Queda explicito el derecho de eliminar cuenta e informacion.
- Consistencia visual con landing y paginas publicas.
